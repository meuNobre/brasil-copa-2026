Olá! Sou João Miguel, desenvolvedor independente. Este projeto foi criado para acompanhar a campanha da Seleção Brasileira na Copa do Mundo de 2026, com bolão e ranking entre torcedores. Os dados são fornecidos pela Football-Data.org e processados por uma API própria.
